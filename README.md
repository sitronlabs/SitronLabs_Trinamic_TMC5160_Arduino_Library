# Trinamic TMC5160 Arduino Library
Arduino library for the Trinamic TMC5160 stepper motor driver.

### Supported interfaces
| Interface | Status |
|-----------|:------:|
| UART | ❌ |
| SPI | ✔️ |

### Disclaimer
Because this library uses bitfields, you might have to add the `-Wno-packed-bitfield-compat` compile flag to remove warnings.
